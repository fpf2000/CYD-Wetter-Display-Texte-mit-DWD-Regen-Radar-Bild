# CYD-Wetter-Display-Texte-mit-DWD-Regen-Radar-Bild

Ziel ist es auf dem CYD ESP32 auch ESP32-2432S028R ein paar lokale Wetter Informationen und zusätzlich den Wasserstand unserer Zisterne anzeigen

In kombination mit dem IOBroker der über das JS Script modul das Regenradar Bild läd und die MQTT werde von einem Shelly GW3000 bereitstellt.

An dem Shelly GW3000 ist eine 7+1 Wetter Station und ein LDS01 Laser Distanz Meter angeschaltet

Die berechnung des Füllstand der Zisterne ist noch nicht abgeschlossen und wird noch angepasst.
